## How the Protocol Works

## Developer Quick Start

```bash
# Connect to testnet
RPC:      https://rpc.presschain.io
Chain ID: 77117002
Token:    PRESS

# Get testnet PRESS
curl https://faucet.presschain.io/api/drip \
  -X POST -H "Content-Type: application/json" \
  -d '{"address": "0xYOUR_ADDRESS"}'

# Bridge health
curl https://bridge.presschain.io/health

# Read a Capsule
curl https://bridge.presschain.io/capsules/1
```

```typescript
import { JsonRpcProvider, Contract } from "ethers";

const provider = new JsonRpcProvider("https://rpc.presschain.io");
// Chain ID: 77117002n

// Read capsule acceptance state
const acceptance = new Contract(
  "0x63D0c0b51a3856a539b36a909517a18BddDb4a32",
  ["function isVotingOpen(uint256) view returns (bool)",
   "function getVoteState(uint256) view returns (uint256,uint256,uint256,uint256,uint256,bool,bool)"],
  provider
);
const open = await acceptance.isVotingOpen(1n);
```
