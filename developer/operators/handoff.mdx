## What You Are Taking Over

PressChain is a sovereign EVM-compatible L1 with a journalism protocol layer. The stack:

- Server `38.83.53.58` — all Next.js apps via systemd, Apache proxy
- Static marketing site at `38.83.53.142` (Vite/React, cPanel)
- An EVM node running PressChain consensus
- Node.js Bridge API middleware
- Mintlify docs at `/home/press/presschain-docs`
- All services managed via `presschain-next@<name>` systemd units

## First Steps After Taking Over

<Steps>
  <Step title="Change admin credentials immediately">
    Update Portal admin password, Bridge API key, and any shared secrets. Do this before anything else.
  </Step>
  <Step title="Rotate the Bridge API key">
    ```bash
    # Generate new key
    openssl rand -hex 32

    # Update on server
    nano /home/press/presschain-bridge/.env
    # Set BRIDGE_API_KEY=new_key

    # Restart Bridge
    sudo systemctl restart presschain-bridge
    # or PM2: pm2 restart presschain-bridge

    # Update all consumers (SYNC plugin, portal env, etc.)
    ```
  </Step>
  <Step title="Verify full stack health">
    ```bash
    bash /home/press/presschain-health.sh
    # (or run the health check commands from Troubleshooting)
    ```
  </Step>
  <Step title="Verify contract addresses">
    ```bash
    curl https://bridge.presschain.io/contracts | python3 -m json.tool
    # Cross-reference with contract registry docs
    ```
  </Step>
</Steps>

## Full Recovery Sequence (Complete Outage)

```bash
# 1. Verify you're on the right server
hostname; whoami

# 2. Check what services exist
systemctl list-units | grep presschain

# 3. Start all app services
for s in portal outlet court proposals ads status docs faucet; do
  sudo systemctl start presschain-next@$s
  echo "Started: $s"
done

# 4. Verify each
for s in portal outlet court proposals ads status docs faucet; do
  echo "$s: $(systemctl is-active presschain-next@$s)"
done

# 5. Restart Apache (for proxy)
/scripts/rebuildhttpdconf && /scripts/restartsrv_httpd

# 6. Verify public access
for url in portal court status docs faucet; do
  code=$(curl -s -o /dev/null -w "%{http_code}" https://$url.presschain.io)
  echo "$url.presschain.io: $code"
done
```
