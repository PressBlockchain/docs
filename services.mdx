## systemd Service Commands

### Restart a Service

```bash
sudo systemctl restart presschain-next@portal
sudo systemctl restart presschain-next@court
sudo systemctl restart presschain-next@status
sudo systemctl restart presschain-next@docs
sudo systemctl restart presschain-next@faucet
```

### Check Status

```bash
systemctl status presschain-next@portal --no-pager -l
systemctl status presschain-next@docs --no-pager -l
```

### View Logs

```bash
journalctl -u presschain-next@portal -n 100 --no-pager
journalctl -u presschain-next@docs -n 100 --no-pager
journalctl -u presschain-next@docs -f   # live follow
```

## Build & Deploy

### Standard App Build

```bash
cd /home/press/PressChain/apps/portal
pnpm build
sudo systemctl restart presschain-next@portal
```

### Full Rebuild (clear .next cache)

```bash
cd /home/press/PressChain/apps/portal
rm -rf .next
pnpm build
sudo systemctl restart presschain-next@portal
```

### Docs Rebuild

```bash
cd /home/press/PressChain/apps/docs
# (Mintlify — no build step needed for dev server)
sudo systemctl restart presschain-next@docs
```
